제목은 , Semantic 하게 작성한다.

### 나열(리스트) 작성하기

#### 순서가 있는 리스트

1. WeMakeO 앱을 받는다.
2. 회원가입한다.
   - 카카오
   - 네이버
   - 페북
3. W카페를 찾는다.
4. 커피를 주문한다.
5. 알림이 오면 픽업하러 간다.

#### 순서가 없는 리스트

- 파이썬 설치하기
- 타이포라 설치하기
- Git 설치하기
